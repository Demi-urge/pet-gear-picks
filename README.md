## Deploy to GitHub Pages

```bash
cd pet-gear-picks
git init
git add .
git commit -m "Initial site — 10 affiliate articles"
git remote add origin https://github.com/Demi-urge/pet-gear-picks.git
git push -u origin main
```

Then enable GitHub Pages in repo Settings → Pages → Source: main branch.
